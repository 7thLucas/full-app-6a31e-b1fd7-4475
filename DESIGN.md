# PawHub Design System

## Colors
- Primary: Warm orange (#FF6B35) — energy, warmth, pet friendliness
- Secondary: Teal (#2EC4B6) — trust, cleanliness, health
- Background: Off-white (#FAFAF8)
- Text: Dark charcoal (#2D2D2D)
- Accent: Soft green (#8BC34A) for health/wellness indicators

## Typography
- Headings: Rounded sans-serif (friendly, approachable)
- Body: Clean sans-serif (readable, professional)
- Tone: Warm and caring, never clinical

## Design Principles
- Card-based layouts for products, services, pets
- Warm illustrations or pet icons
- Clear CTAs (Book Now, Shop Now, Chat with AI)
- Mobile-first responsive design
- Easy navigation between Shop / Vet / Hotel / AI Assistant sections

## Components
- Pet profile cards
- Service booking cards with availability
- Product grid with filters
- AI chat bubble (floating, accessible)
- Admin dashboard with clean data tables and charts
